# 테스트 케이스 (Test Cases)

- 표기: **TC-{영역}-{번호}**, 유형 = 정상 / 경계 / 예외 / 보안
- 우선순위: P1(높음) · P2(보통) · P3(낮음)
- 상태 예시: ✅Pass / ❌Fail / ⏭️Blocked (실제 수행 시 갱신)
- ❌Fail 항목은 결함 리포트([03_defect-report.md](03_defect-report.md))의 DEF-ID와 연결됩니다.
